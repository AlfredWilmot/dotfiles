# OS notes
Using arch.
- install nvim:
    `sudo pacman -S nvim`
- purge existing vim instances installed via pacman:
    `sudo pacman -Rs vim`

- Using [packer.nvim](https://github.com/wbthomason/packer.nvim) for package manage ment)

- Using [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) for file fuzzy-finding
